1.网络请求

请求过程

  • 浏览器向服务器发送请求

分为四个部分

  • 请求网址 - request url
  • 请求方法 - request methods
  • 请求头 - request header
  • 请求体 - request body

F12 查看响应

2.爬虫的概念

模拟浏览器 - 发送请求 - 获取响应

特点：

  • 只能获取客户端展示出的数据
  • 模拟正常操作的用户
  • 获取信息碎片化
  • 爬虫、反爬、反反爬

3.爬虫的作用

  1. 数据采集
  2. 软件测试
  3. 抢票
  4. 网络安全
  5. web漏洞扫描

4.爬虫分类

  • 根据爬取网站的数量分： 通用爬虫、聚焦爬虫（学这个）
  • 根据获取数据的目的分：功能性爬虫、数据增量爬虫

5.爬虫流程

  1. 目标网址
  2. 发送请求，获取响应
  3. 解析响应（有分页等，重复1-3步骤）：jsonpath ，xpath，re
  4. 保持数据：本地json、txt等，数据库

6.网络请求步骤

  1. 电脑（浏览器）：url - www.baidu.com 域名
  2. DNS服务器：ip地址 标注服务器 - 1.1.1.1 IP地址
  3. DNS服务器返回 ip地址 给服务器
  4. 浏览器拿到 ip地址 去访问服务器，返回响应
  5. 服务器返回给我们响应数据，html/css/js/jpg…

实际原理：

  • 一个请求www.baidu.com 只能对应一个数据包（文件）
  • 之后抓包可能会有很多个数据包，共同组成了这个页面
